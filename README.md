# check - Система управления проектами

## Описание

**check** — это веб-приложение для отслеживания задач и управления проектами. 

### Основные возможности

- **Управление проектами** — на главной странице видно все проекты, чтобы никакой не остался без внимания
- **Трекинг задач** — задачи позволят отслеживать прогресс проектов
- **Быстрые заметки** — сохраняйте мимолетные мысли, чтобы потом разобрать
- **Итоги** — каждый вечер смотрите свой прогресс за день
- **Уведомления** — напомним о дедлайнах и важных вещах

## Быстрый старт

### Предварительные требования

- Python 3.10 или выше
- pip (менеджер пакетов Python)

### Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/eledays/check.git
   cd check
   ```

2. **Создайте виртуальное окружение**
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate  # Для Linux/macOS
   # или
   .venv\Scripts\activate  # Для Windows
   ```

3. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```

4. **Инициализируйте базу данных**
   ```bash
   flask --app run.py db upgrade
   ```

5. **Запустите приложение**
   ```bash
   python run.py
   ```

6. **Откройте в браузере**
   ```
   http://127.0.0.1:5000
   ```

## Модель данных

### User (Пользователь)
- `id` — уникальный идентификатор
- `telegram_id` — ID пользователя в Telegram

### Project (Проект)
- `id` — уникальный идентификатор
- `name` — название проекта
- `short_name` — короткое имя (до 16 символов)
- `description` — описание проекта
- `goals` — цели проекта
- `creator_id` — связь с пользователем

### Task (Задача)
- `id` — уникальный идентификатор
- `title` — название задачи
- `status` — статус (TODO, IN_PROGRESS, DONE)
- `project_id` — связь с проектом

### Note (Заметка)
- `id` — уникальный идентификатор
- `content` — содержимое заметки
- `project_id` — связь с проектом
